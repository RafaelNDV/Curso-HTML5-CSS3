UMA RUMA DE TAG IMPORTANTE DO HTML:
 
head -> cabeça

body -> corpo

h1 até h6 -> São título, e existe uma hierarquia

hr -> faz uma linha horizontal até o final da tela

p -> faz um parágrafo

br -> quebra a linha

& -> server para botar uma ruma de símbolo, contando emoji, dolar, euro, copyright, maior que, menor que

&lt e &gt -> menor que e maior que = LESS THAN e GREATER THAN

emoji -> &#x(código do emojipedia)

<img src="(caminho)" alt="(descrição da imagem"> -> para carregar uma imagem

link:favicon -> para botar o favicon no lugar certo

b -> deixa o texto em negrito sem samântica

strong -> deixa o texto em negrito com semântica

i  -> deixa o texto em itálico sem semântica

em -> deixa o texto em itálico com semântica

style -> serve para estilizar que é basicamente usar o CSS3

small -> serve para deixar o texto pequeno

del -> risca o texto com sentido semântico

ins -> sublinha o texto com sentido semântico

sup -> Para inserir coisas do tipo x20+3

sub -> Para inserir coisas do tipo H2O

code -> deixa a escrita mono espaçada e é bom para ler código

q -> deixa o texto entre aspas dando o sentido de ser um citação

abbr -> para botar o sentido das abreviações com sentido semântico

ol type="1" -> faz listas ordenadas com númeração

li -> para botar um item na lista

ul type="disc" -> faz listas não ordenadas

dl -> Lista de definição

dt -> item dessa lista

dd -> descrição do item dessa lista

a href="(site)" target="_blank" rel="external" -> para botar o link, e dá pra soblinar a palavra e talz

audio src="" controls autoplay -> para botar um aúdio no site, com uma caixinha até bacana

video src="(caminho)" controls autoplay

UMA RUMA DE COISA IMPORTANTE DO CSS:

background-color: (); -> muda cor de fundo

color: ();  -> muda a cor

font-family: -> muda a fonte do texto

font-size: ()px; -> muda o tamanho da fonte

background-image: linear-gradient(to (top, bottom, right, left), cores -> pra fazer o gradient

background-attachment: fixed; -> deixa fixo o fundo
